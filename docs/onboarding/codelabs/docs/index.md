# CODELABS - Laboratorios de Código

## ¿Qué son los Codelabs?

**Codelabs** es nuestra plataforma de aprendizaje interactivo con tutoriales prácticos paso a paso para dominar nuevas tecnologías.

- **[Nivel Avanzado](codelabs/advanced.md)** - Arquitecturas complejas, Performance
