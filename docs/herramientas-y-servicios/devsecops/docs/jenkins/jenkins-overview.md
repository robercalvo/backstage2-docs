# Jenkins: Introducción y conceptos básicos

## Documento alojado en GitHub

## ¿Qué es Jenkins?
Jenkins es un servidor de automatización **open source** que permite automatizar tareas relacionadas con la construcción, pruebas y despliegue de software. Es altamente extensible mediante plugins.

### Características principales:
- **Automatización CI/CD**: Integración continua y entrega continua.
- **Extensible**: Miles de plugins disponibles.
- **Multiplataforma**: Funciona en Linux, macOS, Windows.
- **Instalación flexible**: Paquetes nativos, Docker, WAR standalone.

## Instalación
- **Requisitos**: Java Runtime Environment (JRE).
- **Métodos**:
  - Paquetes nativos (apt, yum).
  - Imagen Docker oficial.
  - Archivo WAR ejecutable.

## Conceptos clave:
- **Jobs**: Tareas automatizadas.
- **Pipeline**: Flujo de construcción y despliegue como código.
- **Agentes**: Máquinas donde se ejecutan los jobs.
- **Plugins**: Extensiones para nuevas funcionalidades.
